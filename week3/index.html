      Objects as Parameters to Functions

                              
/*An object literal can be passed as a parameter to a function. This is useful when there are a large number of 
parameters, as it allows the arguments to be provided by name and in any order. This means you don't have to 
remember the order to enter them when invoking a function.*/
function greet({greeting,name,age}) {
  return `${greeting}! My name is ${name} and I am ${age} years old.`;
